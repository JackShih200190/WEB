# 網頁設計
## div練習
```
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>示範div區塊</title>
  </head>
  <body>
    <div>
      <h1>我的程式集</h1>
      <ul>
        <li><a href="br.html">換行練習</a></li>
        <li><a href="超連結.html">超連結練習</a></li>
        <li><a href="article.html">文檔獨立練習</a></li>		
      </ul>
    </div>
  </body>
</html>
```
 ## 超連結
 ```
  <!doctype html>
 <html>
  <head>
    <meta charset="utf-8">
	<title>我的第一次網站</title>
  </head>
  <body>
	<h1>第一次上課</h1>
	<p>今天第一次學習<br>
		我覺得很有興趣</p>
	<ul>
		<li><a href="https://www.youtube.com">測試</a></li>
		<li><a href="https://www.google.com.tw/">網站</a></li>
	</ul>	
  </body> 
 </html>
 ```
